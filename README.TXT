REPORTFLOW - Sistema de Relatorios Automaticos
==============================================

Descricao
---------
Landing page SaaS para sistema de relatorios automaticos.
Conecta Google Sheets, gera PDFs profissionais e envia por email.
Design moderno com tema claro/escuro (Day/Night mode).

Tecnologias
-----------
- HTML5
- CSS3 (variaveis CSS para temas)
- JavaScript vanilla
- Google Fonts (Inter)
- Zero frameworks
- Zero dependencias externas

Como usar
---------
1. Baixe os arquivos index.html, style.css e script.js
2. Coloque na mesma pasta
3. Abra o index.html no navegador
4. Ou use o Live Server no VS Code

Funcionalidades da Landing Page
-------------------------------
- Navbar sticky com toggle de tema
- Hero section com mockup de dashboard
- 6 cards de recursos
- Formulario de configuracao (URL Sheets, email, dia, hora)
- Tabela de precos (3 planos)
- Design responsivo
- Tema claro/escuro com persistencia

Estrutura de arquivos
---------------------
reportflow/
├── index.html          (estrutura HTML)
├── style.css           (estilos e temas)
├── script.js           (logica de tema e formulario)
└── readme.txt          (este arquivo)

Personalizacao
--------------
- Edite style.css para mudar cores (variaveis --bg, --text, --accent)
- Edite index.html para mudar textos, precos e nome da empresa
- Substitua "ReportFlow" pelo nome do seu produto
